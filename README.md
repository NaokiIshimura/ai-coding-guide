ai-coding-guide/

│   ├── CLAUDE.md
├── guides/

## command
### ClaudeCode
CLAUDE.md
```
$ cp claude/CLAUDE.md ~/.claude/CLAUDE.md
```

commands
```
$ mkdir ~/.claude/commands/
$ cp claude/commands/* ~/.claude/commands/
$ ls -l ~/.claude/commands/
```
